## Welcome to Credzu API

Credzu is a consumer protection platform for the credit repair industry. Our API allows partners to integrate with our escrow system, manage transactions, and access credit repair services programmatically.

### Base URL

```
https://api.credzu.com/v1
```

### What you can do

- **Escrow Management** - Create and manage escrow transactions
- **Consumer Data** - Access consumer profiles and credit information
- **Payments** - Handle payment processing and disbursements
- **Webhooks** - Receive real-time notifications on transaction events
